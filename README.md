# AcdnRentCarCN.sql
This project is my first academic study using MySQL. It was developed based on the ADCN Rent Car challenge published in MySQL Magazine, which proposes designing and implementing a complete car rental management system.
The objective of this project is to model the database structure, define the necessary relationships, and create the SQL scripts required to support core rental operations such as vehicle registration, customer management, reservations, and rental tracking.
This repository contains all scripts, tests, and notes produced during the study, serving as a practical learning exercise in database design and MySQL development.
